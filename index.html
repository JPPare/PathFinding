About

  • The first mouse-click on the grid with place a green start cell. The second click will place a dark blue goal cell.
  • After these cells are placed you can use the buttons on the settings bar to clear them, the next click will place them again.
  • Clicking and dragging will allow you to create walls or remove them.
  • During the search blues cells indicate nodes the algorithm has visited. A red cell indicates a node that has been seen and will be returned to later.
  • You can run the maze algorithm to generate a good test for the path finding algorithms.
